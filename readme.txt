Woocommerce watchlist notification plugin is allow users to put products on the watchlist. So that when any similar product will be arrived on the stock users will get notified about that new product which is matched with their keyword.

Basically its for similar product notification. 

== Description ==
Woocommerce watchlist notification is an addon of woocommerce . Allow to add products to the watchlist. Once product will be added to the watchlist user will be able to get notification emails whenever any new similar product will be added to the system.

This plugin focused on ease of integration, ease of use, and extensibility. It is deliberately powerful yet unbelievably simple.

Users can customize button, styling, email description from the setting page. 

Users can manage all the keywords on my-account page. They can search, add, delete keyword according to the need.


== Installation ==

1. Upload the entire **woocommerce-watchlist-notification** folder to the **/wp-content/plugins/** directory.
2. Activate the plugin through the **Plugins** menu in WordPress admin.
3. Go to the watchlist notifications settings page and do the settings according to the need.
